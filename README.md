## :toolbox: Getting Started

### :bangbang: Prerequisites

- Install Node JS in your computer <a href='https://nodejs.org/en/'>HERE</a>

<!-- Run Locally -->

### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/nikhil-pn/URL-Shortner-Node.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
 node index.js
```
